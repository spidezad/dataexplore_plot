dataexplore_plot
===============================

version number: 0.0.1
author: Tan Kok Hua

Overview
--------

Plotting module for use in data exploration for ML and Kaggle

Installation / Usage
--------------------

To install use pip:

    $ pip install dataexplore_plot


Or clone the repo:

    $ git clone https://github.com/spidezad/dataexplore_plot.git
    $ python setup.py install
